# Flowzen---Crowd-management-system
An intelligent crowd monitoring and smart queue management platform that leverages AI-driven people detection, real-time density analysis, and digital tokenization to improve safety, reduce congestion, and optimize public-space operations.

✨ Features
📊 Real-Time Dashboard with live crowd statistics and density insights
🎥 AI-Powered Crowd Monitoring using simulated camera feeds and people detection
📍 Multi-Zone Density Tracking with automatic risk classification (Low, Moderate, High, Critical)
📈 Live Analytics & Trend Visualization for crowd movement and occupancy patterns
🎫 Smart Queue Management with digital token generation and QR-based access
🚨 Intelligent Alert System for overcrowding, threshold breaches, and anomaly detection
🔔 Real-Time Notifications with alert severity levels and status tracking
🔐 Admin Authentication Panel for secure system access and management
⚡ Live Data Updates powered by WebSocket-style real-time communication
📱 Responsive UI optimized for desktop and mobile devices
🏗️ System Modules
Crowd Monitoring Engine – Tracks occupancy across multiple zones.
Queue AI Module – Predicts wait times and manages digital token flow.
Alert Engine – Generates and manages critical, warning, and informational alerts.
Analytics Dashboard – Visualizes density trends and crowd statistics.
Authentication Service – Handles secure admin access and authorization.
